# 黑色星期五用户行为数据分析报告

本项目基于沃尔玛用户购物数据集，对用户的消费行为进行深入分析。通过数据清洗、用户分层、消费偏好分析、关联规则挖掘等方法，探索不同用户群体的消费特征、商品偏好及购买行为规律。

## 技术栈
- 数据处理：Pandas、NumPy
- 数据可视化：Matplotlib、Seaborn
- 关联规则挖掘：mlxtend（Apriori算法）

## 数据集说明
数据集包含用户ID、商品ID、性别、年龄、职业、城市类别、居住时长、婚姻状况、商品类别、购买金额等字段，共计约55万条交易记录，覆盖不同人群的购物行为数据。

## 分析内容
1. 数据清洗与字段映射
2. 不同年龄段的消费能力与商品偏好分析
3. 不同婚姻状况与性别的消费水平与偏好差异
4. 不同城市的人均消费水平对比
5. 前10%高价值用户的特征分析
6. 高价值用户的销售额贡献度分析
7. 不同城市偏好的TOP3商品类别
8. 居住时长与消费金额的关系
9. 不同职业的消费金额排名及偏好商品
10. 购物篮分析：商品关联规则挖掘
11. 年轻群体 vs 中老年群体的消费差异
12. 高频用户的特征画像
13. 最典型高频用户的消费贡献度分析
